Стек проекта:
aiogram + aioschedule + DRF + Redis

Запуск сервера:
python3 manage.py runserver

Запуск бота:
python3 manage.py startbot

Примечание(!)
Команды должны выполняться из каталога RedisApi.


Описание работы бота:
При первом нажатии /start в базе данных на сервере создается запись, содержащая id пользователя и
время его взаимодействия с ботом. При каждом последующим взаимодействии пользователя с ботом в БД
происходит обновление времени.
Одновременно с этим каждые check_delay секунд выполняется запрос к серверу, результатом которого является
список id пользователей, которые взаимодействовали с ботом более offline_time секунд назад. Далее этот список
передается в функцию, которая осуществляет рассылку сообщений пользователям об их длительном оффлайне.
